## Getting Started

To get started, clone the
[aptos-sdk](https://github.com/switchboard-xyz/aptos-sdk) repository.

```bash
git clone https://github.com/switchboard-xyz/aptos-sdk
```

Then install the dependencies

```bash
cd aptos-sdk
pnpm install
```
