## Learn More

Want to explore the Live network? Checkout the
[Switchboard Explorer App](https://app.switchboard.xyz/) to view existing feeds
and when ready, build your own feed!

Checkout the following links to learn more about the Switchboard network
including its technical architecture and supported products.

**Technical Documentation**

- [The Oracle Problem](/the-oracle-problem): Why do we need oracles?
- [Program Architecture](/architecture): Learn how the network is structured and
  the different components.
- [The Data Models](/data): Learn about the different ways to source data
  on-chain.
- [What are TEEs?](/tee): Learn about Trusted Execution Environments (TEEs) and
  how Switchboard leverages them to provide verifiable compute.

**Products**

- [Switchboard Functions](/functions): Learn how Switchboard functions allow
  developers to write their own code and modify their smart contracts state on a
  schedule or on-demand (callbacks).
- [Switchboard Secrets](/secrets): Learn how to use confidential information
  within your function.

:::note

Checkout the [Ecosystem](https://switchboard.xyz/ecosystem) page to see a list
of projects currently using Switchboard!

:::
