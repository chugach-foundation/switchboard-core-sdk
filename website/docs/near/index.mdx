## Getting Started

To get started, clone the
[near-sdk](https://github.com/switchboard-xyz/near-sdk) repository.

```bash
git clone https://github.com/switchboard-xyz/near-sdk
```

Then install the dependencies

```bash
cd near-sdk
pnpm install
pnpm build
```
