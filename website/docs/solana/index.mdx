## Oracle Program

The Oracle Program was deployed to Solana mainnet in January of 2022.
The Oracle Program supports Data Feeds and Verifiable Randomness Functions (VRF) in a permissioned and permissionless manner.

## The Attestation Program

The Attestation Program supports Switchboard Functions, Secrets, and legacy Data Feeds.
Data Feeds on an oracle queue with the sgx setting enabled enforces oracles to heartbeat weekly with an
attestation report signaling that they are running within a secure enclave.

## SDKs

### Javascript

The npm package [@switchboard-xyz/solana.js](/api/solana.js) can be used to build and manage Switchboard accounts on Solana.

### Rust

The rust crate [switchboard-solana](https://crates.io/crates/switchboard-solana) can be used on-chain and off-chain.
The previous crate [switchboard-v2](https://crates.io/crates/switchboard-v2) is only compatible with Switchboard V2 accounts and does not support Switchboard Functions.

### CLI

The [Switchboard CLI](/api/cli) can be used to help debug your Switchboard deployments with a command line interface or be used in automation scripts for custom alerting.
